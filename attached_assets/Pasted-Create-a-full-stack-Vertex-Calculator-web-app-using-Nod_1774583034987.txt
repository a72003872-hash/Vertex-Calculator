Create a full-stack Vertex Calculator web app using Node.js and Swiss Ephemeris (swisseph).

GOAL:
Build an accurate astrology tool that calculates the Vertex sign based on birth date, time, and location using Swiss Ephemeris (not approximations).

---

## FRONTEND REQUIREMENTS:

Create a clean, minimal form (NO header, footer, hero section).

Fields required:
1. Your Name 

   * Input type:  Enter Name

1. Birth Date

   * Input type: date

2. Birth Time

   * Input type: time (24-hour format)

3. Birth City

   * Text input with autocomplete dropdown
   * Use OpenStreetMap Nominatim API:
     https://nominatim.openstreetmap.org/search?format=json&q=

4. Submit Button:

   * Label: "Calculate Vertex"

5. After result:

   * Show result section below form
   * Include "Calculate Again" button (resets form)

## API ROUTE:

POST /calculate-vertex

Request body:
{
date: "YYYY-MM-DD",
time: "HH:MM",
lat: number,
lon: number
}

Response:
{
vertex_degree: number,
vertex_sign: string
}
